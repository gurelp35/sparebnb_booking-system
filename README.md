The system demonstrates Object-Oriented Programming principles and simulates a real-world booking platform with multiple accommodation types and booking management functionality.

## Features

- Accommodation management (Hotel, Flat, Villa)
- Booking creation and management
- User management
- Modular class-based architecture
- Object-Oriented Programming (OOP)

## Technologies Used

- Java
- Object-Oriented Programming
- GitHub version control

## Key Classes

- BookingManager.java – manages booking
