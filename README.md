# SpareB&B Booking System (Java)

SpareB&B is a structured object-oriented booking system developed in Java.  
The application models accommodation listings, customer bookings, and validation logic using core OOP principles.

---

## Project Overview

This system simulates a simplified accommodation booking platform.  
It applies object-oriented design to manage properties, reservations, and user interactions in a modular and maintainable way.

The implementation demonstrates:

- Class-based system architecture
- Encapsulation and data abstraction
- Inheritance and method overriding (where applicable)
- Validation logic for booking rules
- Structured control flow
- Clean separation of responsibilities

---

## System Design

The application includes:

- Property / Listing modelling  
- Booking entity management  
- Customer data handling  
- Validation rules for reservation constraints  
- Structured interaction logic  

The design emphasises maintainability, scalability, and clarity of object responsibilities.
